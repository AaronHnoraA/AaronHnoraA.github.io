# Slides Kind Demo

This note has no `id` field in its meta block.

It is still a usable Aaronnote note because `kind: slides` is enough to opt into the slide assets.

# How Slides Split

Each top-level heading starts a new slide.

- Keep slide content under one `# Heading`
- Use regular Markdown blocks
- Use `---` if you want a manual split without a new heading

# Controls

In a published page, use the arrow keys, page keys, Home, End, or the buttons.

Inside the editor, use the buttons or `Alt` plus arrow keys so normal text editing still works.

# Why This Exists

The `slides` kind is intentionally small.

It proves the lazy-loading hook works now, while leaving space for a richer deck mode later.
